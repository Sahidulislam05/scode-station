## üìñ Overview

This is a **Hybrid Tech Ecosystem** **_"Scode Station"_** platform that combines two major business models:

1.  **Service Agency (Like Brain Station 23):** Providing high-end software development services (Web, App, Automation).
2.  **EdTech Academy (Like 10 Minute School):** offering seminars, workshops, and training to groom future tech talent.

Built with the latest **Next.js 16 (App Router)**, fully typed with **TypeScript**, and styled using **Tailwind CSS + Shadcn UI**.

---

## üõ† Tech Stack

- **Framework:** Next.js 15 (App Router, Turbo)
- **Language:** TypeScript
- **Styling:** Tailwind CSS, Shadcn UI, Framer Motion
- **Icons:** Lucide React, React Icons
- **Database:** MongoDB (via Mongoose ODM)
- **Authentication:** Custom JWT (HttpOnly Cookie + LocalStorage Strategy)
- **State Management:** React Context API + Zustand (optional)
- **Form Handling:** React Hook Form + Zod

### üè¢ Agency Module (Client Facing)
